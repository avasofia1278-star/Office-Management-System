# Office Management System

## Description
This is a console-based Office Management System developed in C++ using Object-Oriented Programming (OOP).

## Features
- Add Employee
- Search Employee
- Update Employee
- Delete Employee
- Display Employee Records

## Technologies Used
- C++
- Object-Oriented Programming (OOP)

## How to Run
1. Download the project.
2. Open it in Code::Blocks or Visual Studio.
3. Compile and run `main.cpp`.
